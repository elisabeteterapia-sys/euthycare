# Quality Criteria — Roteiro de Metáforas

## Critérios Globais (aplicam a todos os formatos)

| Critério | Peso | Limiar mínimo |
|----------|------|---------------|
| Clareza da metáfora | Alto | 7/10 |
| Coerência do ângulo emocional | Alto | 7/10 |
| Adequação ao público (terapeutas/Euthycare) | Alto | 7/10 |
| Hook (primeiros 3 segundos) | Crítico | 8/10 |
| CTA presente e específico | Médio | 6/10 |
| Tom de voz alinhado ao ângulo | Alto | 7/10 |
| Ausência de promessas terapêuticas indevidas | Crítico | 9/10 |

**Hard rejection triggers:**
- Qualquer promessa de cura ou resultado terapêutico garantido → REJEITAR
- Metáfora que reforça estigma ou patologiza → REJEITAR
- Hook genérico que não prende em 3 segundos → REJEITAR
- Ausência de CTA → REJEITAR

## Critérios por Formato

### Instagram Reels (~1 min)
- [ ] Hook visual + falado em 0-3s que para o scroll
- [ ] Duração entre 45-90 segundos
- [ ] Script inclui timestamps para cortes
- [ ] Legendas/subtítulos especificados
- [ ] Guia CapCut com: cortes, texto overlay, música sugerida
- [ ] Caption com hook em 125 caracteres
- [ ] 5-15 hashtags relevantes
- [ ] CTA específico (não genérico "sigam")
- [ ] Proporção 9:16 especificada

### Instagram Feed (Carrossel)
- [ ] Formato de carrossel escolhido e justificado
- [ ] 8-10 slides com estrutura correta
- [ ] Cada slide: 40-80 palavras
- [ ] Slide de capa para o scroll
- [ ] Alternância de fundos (claro/escuro/acento)
- [ ] Caption com hook em 125 caracteres
- [ ] CTA no slide final
- [ ] 5-15 hashtags

### Instagram Stories (~1 min)
- [ ] 5-8 stories em sequência
- [ ] Cada story: texto overlay claro e legível
- [ ] Progressão narrativa da metáfora do início ao fim
- [ ] Story final com CTA (enquete, link ou resposta)
- [ ] Sugestão de stickers/GIFs adequados
- [ ] Direção de cor de fundo por story

### YouTube Shorts (~1 min)
- [ ] Hook em 0-2s impossível de ignorar
- [ ] Duração 30-60 segundos
- [ ] Um só ponto central — sem desvios
- [ ] Loop design: final conecta ao início
- [ ] Texto overlay durante todo o Short
- [ ] Título curto e curioso
- [ ] #Shorts na descrição
- [ ] Guia CapCut incluído

### YouTube Script (~5 min)
- [ ] Título 50-70 caracteres com curiosity gap
- [ ] Conceito de thumbnail definido
- [ ] Hook estabelece premissa em 30 segundos
- [ ] 3-4 seções no corpo com subheadings
- [ ] Pattern interrupts a cada 30-60 segundos marcados
- [ ] Clímax/aha moment em ~60-70% do vídeo
- [ ] Timestamps completos
- [ ] Descrição com primeiras 2 linhas otimizadas
- [ ] Guia CapCut com cortes e recursos sugeridos
- [ ] 5-10 tags

## Rubrica de Pontuação

| Faixa | Descrição | Ação |
|-------|-----------|------|
| 9-10 | Excelente — pronto para publicar | APPROVE |
| 7-8 | Bom — atende todos os critérios | APPROVE |
| 5-6 | Adequado com ressalvas | CONDITIONAL APPROVE |
| < 5 em qualquer critério crítico | Falha grave | REJECT |
| < 7 na média geral | Não atinge o mínimo | REJECT |
